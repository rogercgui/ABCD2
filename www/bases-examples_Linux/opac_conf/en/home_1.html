What's New in OPAC ABCD

Version 1.2.0-beta

Release Date: October 06, 2025

  Implemented "Did you mean?"

  • Optimized Term List Generation: The system now offers two ways to create the suggestion list:
    • Fast Method (ifkeys): Ideal for very large databases, exporting the entire term dictionary almost instantly.
    • Controlled Method (WXIS): A more detailed process that generates a higher quality term list based on indexing rules, ideal for more accurate results.

  Alphabetical Index Improvements

  • Dynamic Dictionary Management: Greater control over the fields that make up the static term list.
  • Editing Flexibility: The interface for adding and removing index configuration lines is now more flexible and intuitive.
  • New Action Buttons: Buttons have been included to trigger the new term list generation methods directly from the configuration screen.

  Security

Cloudflare's invisible CAPTCHA feature has been added. To implement, simply get the free keys at Cloudflare Turnstile and add them in the General > Parameters > Security menu.

  More Secure Search URL

The search script name (`buscar_integrada.php`) has been removed from the URL, making user navigation cleaner and more secure.

  Analytics

Search term logs are now stored in monthly files (year-month), making the analysis page load much faster and simplifying log management.

  Home Page

The home page manager has been fixed and improved, now allowing you to save and switch between multiple HTML content page templates.

  Database Configuration

The database configuration screen has been improved, now displaying a diagnostic checklist of essential files and allowing for quick editing of the public name and description.

  Record Toolbar

  • The toolbar configuration is now dynamic, allowing individual buttons to be enabled/disabled.
  • New Feature: Added support for "Permalink," which generates a direct link to a specific record.

  General Improvements

  • Fixed a bug in the pagination numbering of search results.
